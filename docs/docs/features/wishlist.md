# Wishlist

The **Wishlist** page (header link or `/wishlist`) is your want list — cards you're looking to
acquire, kept separate from the cards you own.

- **Add from a card page** — every card's detail page has an **☆ Add to wishlist** button that
  toggles to **★ On wishlist — remove**.
- **Add a deck's missing cards** — on a [deck](decks.md) page, **☆ Add missing to wishlist** adds
  every card the deck still needs (matched cards only), tagged with a note like *for Goblins*. The
  desired quantity is the number you're short, and re-adding the same deck won't double it.
- **Estimated cost** — the page totals the current Scryfall USD price across everything on the
  list, so you can see what finishing your wants would run.

Each entry stores a desired **quantity** and an optional **note**. Adding a printing that's already
listed raises its quantity to the larger of the two rather than stacking, so imports stay tidy.

On the read-only public demo the wishlist is viewable but can't be changed.
