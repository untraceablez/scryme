# Advanced search

Don't know the [search syntax](syntax.md) yet? The **Advanced** search page (header link or
`/advanced`) is a form-based builder.

Fill in only the fields you need — name / oracle text / type, colors and color identity (with
*including* / *exactly* / *at most* modes), mana value, power / toughness / loyalty ranges, rarity,
set, format legality, keywords, price range, and year range — plus the scope and sort order.

As you fill it in, a **Generated query** preview shows the exact Scryfall-syntax query it will run.
Submitting takes you to the normal results page with that query in the search bar — so the advanced
form is also a way to **learn the syntax by example**: build a query with the form, then tweak the
text directly next time.
